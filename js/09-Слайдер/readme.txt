----------
Разбивка задачи на 3 уровня:
- Визуальный интерфейс (View)
- Логика (Logic)
- Данные (Data)

На уровне Data реализовано 2 метода получения данных - вручную и через Ajax
Передача данных с уровня Data до уровня View при первоначальной инициализации реализована двойным Колбэком (без промисов)
