Калькулятор Версия 2
Калькулятор более приближенный к реальному калькулятору.
Ввод чисел с кнопок
Вывод результатов в реальном времени (пишется сам пример)
Вывод результата по нажатию на знак "="
Сам очищает поле вывода, если начинаем решать другой пример.

Из недостатков:
не нравится метод вывода результатов в реальном времени
out(num1);
out(num1 + ' ' + operation);
out(num1 + ' ' + operation + ' ' + num2);
out(num1 + ' ' + operation + ' ' + num2 + ' = ' + result);

Возможно есть варианты оптимизации.
Вариант с конкатенацией результата
resultFiled.innerHTML += outContent
некорректно работает, если вводим не онозначное число, а двузначное
