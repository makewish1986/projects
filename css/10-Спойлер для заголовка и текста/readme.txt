Применение :before и :after для создания стрелки
С помощью jQuery решаем вопрос с открытием/закрытием спойлера

Примечание:
Реализовано 2 вида спойлера.
Заказчик может попросить, чтобы при каждом следующем нажатии на заголовок, остальные открытые ранее тексты оставались открытыми, а может попросить обратное - чтобы все остальные ранее открытые закрывались, кроме текущего.
Это реализовано с помощью класса '.one'.
При реализации проекта мы просто в html пропишем один из двух видов спойлера
<div class="block"> - оставялем все открытыми
<div class="block one"> - закрываем все открытые, кроме текущего
