Примечание:
1) Треугольники, мы строим не через width/height, а через различные комбинации border. Отсечение определенных сторон происходит при использовании прозрачного фона.
В CSS используются нулевые параметры width и height. Они задаются, чтобы при добавлении контента в этот блок, трегуольник не разъезжался.
2) Трапеция, получается, как разновидность построения треугольника, только модифицируется через ненулевое значение width
