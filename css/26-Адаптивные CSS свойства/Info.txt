Адаптивное (отзывчивое) CSS свойство. Как быстро адаптировать элементы.
Повторение урока
https://www.youtube.com/watch?v=eaOAY0vIB4U

Суть метода, что нужные CSS свойства сами по себе начинают адаптироваться при изменении ширины экрана.
Используется пропорцинальное вычисление такого свойства на промежутке от минимального значения к максимальному.
Например изменяем свойство padding.
У нас есть ограничивающий контейнер в 1200px.
На этой ширине экрана указываем padding: 50px. Это получается максимальный padding.
Также нам нужно знать минимальный padding на минимальном разрешении 320px.
Промежуточные значения padding рассчитываются пропорционально по формуле:
padding: calc(20px + (50 - 20) * ((100vw - 320px) / (1200 - 320)));


В уроке также показывается, как реализовывать эту формулу, когда нет ограничивающего контейнера и для случаев, когда нам нужно применять отзывчивые CSS свойства на разрешениях больших, чем нужно.
