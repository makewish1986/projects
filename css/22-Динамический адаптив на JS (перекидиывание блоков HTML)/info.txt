Динамический адаптив. Скрипт перекидывает блок HTML из одного места в другой

Повторение урока:
https://www.youtube.com/watch?v=QKuMr575vlQ&list=WL&index=1

Для работы скрипта нужно в HTML внутри тега перекидываемого блока указывать атрибут data-da="where, position, when"
where - куда перекидываем блок (в какого будущего родителя)
position - на какое место в блоке назначение будет вставлен перекидываемый блок
when - при каком разрешении это будет происходить
Пример: data-da="item, 2, 640"

В скрипте есть возможность работать с моедлью mobile-first, но для этого нужно изменить два параметра в самом скрипте (в скрипте есть комментарии в нужных местах)

В качестве position можно также указывать значения "first" и "last"

В данном примере перекидываются:
- блок "Я Коля" в блок content__column-river на разрешении меньше, чем 900
- блок "Я Ваня" в блок content__column-garden на разрешении меньше, чем 800
