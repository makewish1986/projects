Функционал спойлера.
Для включения аккордеона добавляется атрибут data-one-spoiler к оболочке спойлера.
Реализован функционал работоспособности спойлера для разных значений брейкпоинтов. Для этого заполняются значения атрибута data-spoilers. Например:
data-spoilers="800,max"
data-spoilers="650,min"

Для обычного спойлера атрибут data-spoilers не заполняется.
